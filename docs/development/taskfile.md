# Taskfile Reference

All tasks available via `task --list`. Key variables: `ZONE_TYPE`, `AUTH` (default: `oidc`), `DOMAIN`.

## Commands

| Command | Description |
|:--------|:------------|
| `task setup ZONE_TYPE=public DOMAIN=x.com` | Generate values file (OIDC mode) |
| `task setup AUTH=static ZONE_TYPE=public DOMAIN=x.com` | Generate values file (static mode) |
| `task deploy` | Deploy/update all zones with existing values files |
| `task deploy ZONE_TYPE=public` | Deploy/update only public zone |
| `task down` | Uninstall all + delete values files |
| `task down ZONE_TYPE=public` | Uninstall only public zone |
| `task status` | Show all Helm releases and pod status |
| `task logs` | Show webhook logs (public) |
| `task logs ZONE_TYPE=private` | Show webhook logs (private) |
| `task logs:dns` | Show external-dns logs |
| `task examples` | Deploy example Services for validation |
| `task examples:clean` | Remove example resources |

## Developer commands

| Command | Description |
|:--------|:------------|
| `task build` | Build and push dev image (`latest` tag) |
| `task test:e2e` | Run e2e acceptance tests |
| `task release TAG=v0.1.0` | Full release: build + push + tag + GitHub release |

## Typical workflows

```bash
# First time setup
task setup ZONE_TYPE=public DOMAIN=example.com
task setup ZONE_TYPE=private DOMAIN=internal.example.com

# Daily development
task build && task deploy

# Update only public zone
task build && task deploy ZONE_TYPE=public

# Switch auth mode
task down ZONE_TYPE=public
task setup AUTH=static ZONE_TYPE=public DOMAIN=example.com
task deploy ZONE_TYPE=public
```

## Variables

| Variable | Default | Description |
|:---------|:--------|:------------|
| `ZONE_TYPE` | _(empty = all)_ | Target zone type (`public` or `private`) |
| `AUTH` | `oidc` | Authentication mode (`oidc` or `static`) |
| `DOMAIN` | | Domain to manage (required for setup) |
| `TAG` | git describe | Image tag for release |

## Generated files

| File | Generated by | Purpose |
|:-----|:-------------|:--------|
| `deploy/helm/values-<zone>.local.yaml` | `task setup` | Instance-specific Helm values (gitignored) |
